# ahmedabad-brts-ticket-system

This project is a console-based simulation of the Ahmedabad Janmarg BRTS ticketing system, developed using Java as part of my Semester 1 programming coursework. The application allows users to register, log in, view available routes, select stations, calculate travel distance, and generate a ticket with fare details.

It was built to practice Object-Oriented Programming (OOP) concepts such as:

> Classes and Objects
> Inheritance
> Methods and Constructors
> Arrays and 2D Arrays
> Input validation using loops
